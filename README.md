# All Weather Strategy

This repository contains the simplified `yfinance` demo version for GitHub and Streamlit Cloud.

## Online Demo

- Streamlit app: https://all-weather-strategy.streamlit.app/

## Run Locally

- Open `app.py` in PyCharm and run it.
- Or run `python app.py` / `streamlit run app.py` from this folder.

## Included Files

- `app.py`
- `all_weather_strategy/`
- `data/offline/`
- `requirements.txt`
- `SIMHEI.TTF`
